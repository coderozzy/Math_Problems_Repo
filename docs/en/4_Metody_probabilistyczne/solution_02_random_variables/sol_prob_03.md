## Given

$U = 15000X + 10000\sqrt{X}$

| $x_i$ | 2 | 3 | 4 | 5 |
|---|---|---|---|---|
| $p_i$ | 0.10 | 0.25 | 0.40 | 0.25 |

## Solution

Calculate $u_i$ for each $x_i$:

- $x=2$: $u = 30000 + 10000\sqrt{2} = 30000 + 14142.1 = 44142.1$
- $x=3$: $u = 45000 + 10000\sqrt{3} = 45000 + 17320.5 = 62320.5$
- $x=4$: $u = 60000 + 10000\sqrt{4} = 60000 + 20000 = 80000$
- $x=5$: $u = 75000 + 10000\sqrt{5} = 75000 + 22360.7 = 97360.7$

## Probability Function of $U$

| $u_i$ | 44142.1 | 62320.5 | 80000 | 97360.7 |
|---|---|---|---|---|
| $p_i$ | 0.10 | 0.25 | 0.40 | 0.25 |

$$\boxed{E(U) = 44142.1(0.10) + 62320.5(0.25) + 80000(0.40) + 97360.7(0.25)}$$

$$= 4414.2 + 15580.1 + 32000 + 24340.2 = \boxed{76334.5}$$
