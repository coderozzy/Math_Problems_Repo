## Given

$P(A) = 0.6$, $P(B) = 0.7$, $P(A \cup B) = 0.8$

## Solutions

**1.** $P(A') = 1 - P(A) = 1 - 0.6$

$$\boxed{P(A') = 0.4}$$

**2.** By inclusion-exclusion: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$

$$0.8 = 0.6 + 0.7 - P(A \cap B)$$

$$\boxed{P(A \cap B) = 0.5}$$

**3.** $P(A' \cap B')$ = probability that neither event occurs.

By De Morgan: $A' \cap B' = (A \cup B)'$

$$P(A' \cap B') = 1 - P(A \cup B) = 1 - 0.8$$

$$\boxed{P(A' \cap B') = 0.2}$$
